# 微信小程序客户端

本参考文献记录了Supabase的同构微信小程序客户端`supabase-wechat-stable-v2`中的每个对象和方法。

通过使用 `supabase-wechat-stable-v2` 你可以实现如下功能：

* 与你的Postgres数据库交互
* 听取数据库变化
* 建立登录和用户管理功能
* 管理大文件
